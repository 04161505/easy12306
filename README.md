### 设计思路简介

1. 借用百度提供的光学字符识别功能，来教我的文字识别器——卷积神经网络。
1. 让文字识别器来教我的图片分类器——统计学专家，看，这张验证码中有打字机哦，这张里也有...
1. 让图片分类器反过来教我的文字识别器，看，这张噪声满满的文字图表示的一定就是这以下八张图片中的某一种。
